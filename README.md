# Scarecrow COPR

The .spec files to allow building Scarecrow on Fedora COPR, to allow easier redistribution of Scarecrow.

The .spec files are edited from the ones avaible at Fedora 33's repos (Fedora 33 is the last version of Fedora with GNOME 3).

# NOTE
If you are installing Scarecrow on your Fedora install with GNOME, IT WILL CONFLICT, as we didn't rebrand any of the actual files of the packages.
